Partie 2 (cliquer pour afficher / cacher)

  1. Réalisez la mise en page suivante grâce à CSS Grid. Cette grille est constituée de 4 colonnes et 5 lignes. Vous n'avez pas à être iso à 100 %, l'idée est avant tout de prendre en main CSS Grid. Toutes les images se trouvent dans le dossier images-partie-1/

    Ici le plus simple n'est pas d'utiliser la balise <img> mais plutôt la propriété CSS background-image et pour la taille du background, on utilisera background-size

    • Accéder à la documentation de background-image
    • Accéder à la documentation de background-size

    Pour la partie CSS Grid, vous avez les liens suivants :

    • Voir documentation de grid-column-start (ouverture dans un nouvel onglet)
    • Voir documentation de grid-column-end (ouverture dans un nouvel onglet)
    • Voir documentation de grid-row-start (ouverture dans un nouvel onglet)
    • Voir documentation de grid-row-end (ouverture dans un nouvel onglet)
    • Voir documentation de gap (ouverture dans un nouvel onglet)

    N'oubliez pas d'ajouter la déclaration CSS display: grid sur le conteneur.
