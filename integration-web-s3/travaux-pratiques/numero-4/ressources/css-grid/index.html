    Vous n'avez pas à être iso à 100 %, l'idée est avant tout de prendre en main CSS Grid. Toutes les images se trouvent dans le dossier images-partie-1/.

    N'oubliez pas d'ajouter la déclaration CSS display: grid sur le conteneur. Et d'importer un CSS reset comme celui d'Eric Meyer pour réinitialiser vos styles.

    Ici le plus simple n'est pas d'utiliser la balise <img> mais plutôt la propriété CSS background-image et pour la taille du background, on utilisera background-size. Enfin pour déplacer le background-image, vous pouvez utiliser la propriété background-position.

    • Accéder à la documentation de background-image - ouverture dans un nouvel onglet
    • Accéder à la documentation de background-size - ouverture dans un nouvel onglet
    • Accéder à la documentation de background-position - ouverture dans un nouvel onglet

    Si vous souhaitez utiliser la balise <img>, il faut impérativement appliquer les déclarations suivantes sur vos images. Sinon, elles déformeront vos grilles. 

    img {
    max-width: 100%;
    min-width: 100%;
    max-height: 100%;
    min-height: 100%;
    object-fit: cover;
}
