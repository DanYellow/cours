Pré-consignes de la pratique - facultatives (cliquer pour afficher / cacher)

  1. Créez un nouveau fichier html (avec la structure de base d'un site web. Rappel, vous avez le raccourci "html:5")
  2. Copiez la balise <main> (et son contenu) dans le fichier HTML que vous venez de créer (si elle existe)
  3. Copiez les balises <link> présentes dans le <head> dans le fichier html que vous venez de créer
  4. Sauvegardez-le et commencez votre TP dedans

Consignes de la pratique

Partie 1 (cliquer pour afficher / cacher)
  1. Accédez au site icomoon (voir encart "Afficher / cacher étapes pour gérer les icônes")
    • Accéder au site icomoon.io (nouvel onglet)
  2. Récupérez les icônes sur icomoon pour les réseaux suivants : Facebook, Twitter, Discord, Slack, Youtube, Github (voir encart "Afficher / cacher étapes pour gérer les icônes")
Afficher / cacher étapes pour gérer les icônes
    • A : Permet d'importer des icones personnalisées. Image au format .svg ou un ensemble d'icônes au format .json
    • B : Permet de sélectionner des des icônes proposées par icomoon
    • C : Boite à outils
    • A : Affiche les préférences de notre iconfont
    • B : Affiche le contenu de notre iconfont
      • B1 : icône
      • B2 : nom de l'icône (modifiable) - Qui sera utilisé comme nom de classe
      • B3 : valeur du caractère
    • C : Permet de télécharger notre iconfont (et tout ce qui va avec)
    • D : Permet de retourner à l'étape précédente
  1. (Désarchivez et) Déplacez le contenu de l'archive dans votre dossier de travail
  2. Liez le fichier .css à votre fichier de travail
  3. Affichez les icônes au bon endroit
    • Pensez à flexbox pour bien aligner les icônes au texte
    • Pensez à la balise <span> pour mettre les classes associées aux icônes

Partie 2 (cliquer pour afficher / cacher)
  1. Ajoutez des transitions au hover (couleur, rotation, échelle...) sur les icônes
    • Voir la documentation de la pseudo-classe ":hover" (nouvel onglet)
    • Voir la documentation des transitions css (nouvel onglet)

Partie 3 (cliquer pour afficher / cacher)
  1. Téléchargez l'icône de votre choix au format svg sur le site iconmonstr (désactivez votre bloqueur de pub)
    • Accéder au site iconmonstr (ouverture dans un nouvel onglet)
  2. Téléchargez une icône d'envoi de votre choix au format svg sur le site iconmonstr (désactivez votre bloqueur de pub) ou svgrepo. Les mots-clés "arrow", "send" ou "plane" vous aiderons à trouver l'icône appropriée
    • Accéder au site iconmonstr (ouverture dans un nouvel onglet)
    • Accéder au site svgrepo (ouverture dans un nouvel onglet)
  3. Uploadez le fichier selection.json que vous avez généré tout à l'heure sur le site icomoon ainsi que les fichiers svg que vous avez téléchargé
    • Accéder au site icomoon.io (nouvel onglet)
  4. Sélectionnez les icônes à utiliser
  5. Regénérez votre police
  6. Réimportez-la dans votre projet
  7. Utilisez les nouvelles icônes dans votre projet notamment pour le bouton Envoyer
