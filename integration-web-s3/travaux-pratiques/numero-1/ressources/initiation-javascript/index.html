Pré-consignes de la pratique - facultatives (cliquer pour afficher / cacher)

  1. Créez un nouveau fichier html (avec la structure de base d'un site web. Rappel, vous avez le raccourci "html:5")
  2. Sauvegardez-le et commencez votre TP dedans

Consignes de la pratique

Partie 1 (cliquer pour afficher / cacher)
  1. Créez un fichier javascript (extension ".js") et liez ce fichier à votre fichier HTML dans la balise <head>. Exemple de code pour importer un fichier javascript (adapter le chemin et le nom de fichier en fonction). 
                            <script src="ressources/scripts.js" defer></script>

    L'attribut 'defer' permet de charger notre fichier javascript après que la partie HTML a été chargée.

  2. Créez deux variables, une contenant un entier et l'autre une chaîne de caractères
    • Voir documentation des variables javascript (ouverture dans un nouvel onglet)
  3. Affichez leur contenu dans la méthode console.log()
    • Voir documentation de la méthode console.log() (ouverture dans un nouvel onglet)

Partie 2 (cliquer pour afficher / cacher)

Comme tout langage de programmation, il est également possible en javascript d'écrire des fonctions, le mot-clé est function. Ensuite il faut nommer sa fonction, et définir ses paramètres. Exemple : 

function maPremiereFonction(param1, param2) {
    return param1 + param2;
}
maPremiereFonction(45, 20);

Dans le code ci-dessus, on définit une variable nommée "maPremiereFonction" prenant deux paramètres et retourne leur addition. Puis on appelle notre fonction avec comme valeurs les nombres 45 et 20. Ca veut donc dire que, dans la fonction "maPremiereFonction", les variables "param1" et "param2" ont pour valeur 45 et 20 respectivement. C'est très simple mais c'est très bien pour commencer. A noter que les opérateurs mathématiques de base sont les mêmes qu'en PHP :

  • + : addition
  • - : soustraction
  • * : multiplication
  • / : division

Pour les commentaires, pour rappel, il vous suffit de préfixer la ligne de code avec //. Cette ligne ne sera pas interprétée. Attention : Comme le HTML et le CSS, les commentaires en javascript sont accessibles à l'utilisateur final. Par conséquent : ne mettez pas de données sensibles dedans.

  1. Réutilisez la fonction ci-dessus et affichez le résultat grâce à la méthode console.log()
    • Vous pouvez stocker le retour de la fonction dans une variable ou appeler directement la fonction dans la méthode console.log().
    • Voir documentation de la méthode console.log() (ouverture dans un nouvel onglet)
    • Voir documentation des fonctions (ouverture dans un nouvel onglet)
  2. Créez les fonctions suivantes :
    • Prend en paramètre un entier et le multiplie par lui-même
    • Prend en paramètres deux chaînes de caractères et souhaite bonjour à cette personne
      • Pour ce faire, il faudra concaténer une chaîne de caractères. Documentation sur la contacténation de chaînes de caractères (ouverture dans un nouvel onglet)
    • Prend en paramètres deux entiers, les multiplie et retourne toujours un produit strictement positif
    • Prend en paramètres deux entiers qui les multiplie sans utiliser l'opérateur de multiplication

Ces fonctions doivent impérativement retourner un résultat (mot-clé"return") et doivent voir leur résultat être affiché dans un console.log().

Exemple :

// Définition de la fonction
function maFonction() {
    return "hello";
}
// Appel de la fonction. N'oubliez pas les parenthèses, 
// indispensables pour appeler la fonction
console.log(
    maFonction();
)

N'oubliez pas de nommer vos fonctions avec des noms clairs et explicites tout comme les paramètres de vos questions.

Partie 3 (cliquer pour afficher / cacher)

Créez les fonctions suivantes :

  1. Prend en paramètre une chaîne de caractères et retourne "Bonjour " + le nom du département dispensé à l'IUT de Sarcelles. Si le département n'est pas proposé à l'IUT de Sarcelles, la fonction doit retourner "Bonjour mais ce département n'est pas proposé à l'IUT de Sarcelles"
  2. Prend en paramètre un nombre entier et qui retourne une chaîne de caractères (ces chaînes sont là à titre indicatif, vous pouvez adapter en fonction de vos besoins) :
    • "Vous êtes un nourrisson" si la valeur est inférieure à 1
    • "Vous êtes un enfant" si la valeur est comprise entre 2 et 12
    • "Vous êtes un adolescent" si la valeur est comprise entre 13 et 17
    • "Vous êtes un adulte" si la valeur est comprise entre 18 et 60
    • "Vous êtes une personne agée" si la valeur n'est dans aucune des bornes suscitées ou n'est pas une valeur correcte

Ces fonctions doivent impérativement retourner un résultat (mot-clé"return") et doivent voir leur résultat être affiché dans un console.log().

Exemple :

// Définition de la fonction
function maFonction() {
    return "hello";
}
// Appel de la fonction. N'oubliez pas les parenthèses, 
// indispensables pour appeler la fonction
console.log(
    maFonction();
)

Partie 4 (cliquer pour afficher / cacher)
  1. Ajoutez un bouton (balise <button type="button">) et au clic appelez une des fonctions que vous avez crée précédemment (ou une nouvelle)
    • Récupérer un élément dans la page HTML (document.querySelector)
    • Ajouter un évènement sur un élément
    • Voir documentation de la balise <button> (ouverture dans un nouvel onglet)
    • Voir documentation de l'évènement "click" (ouverture dans un nouvel onglet)

    Note : Rappelez-vous que le deuxième paramètre de "addEventListener" est un pointeur de fonction. Si vous avez besoin d'afficher un console.log() lors de l'appel, il faudra faire une fonction qui appelle une autre fonction. Par exemple : 

    // On récupère le premier élément qui a la balise <selecteur>
const cible = document.querySelector("selecteur");
// Sur cet élément on lie un évènement "click"
// grâce à la méthode "addEventListener"
cible.addEventListener('click', () => {
    console.log(maFonction());
});
  2. Créez plusieurs balises <p> avec du texte et la même classe CSS. Puis, grâce à la propriété "textContent", faites changer le contenu textuel de ces balises lorsqu'on clique sur un bouton (vous pouvez réutiliser le même ou un créer un nouveau)
    • Voir documentation la propriété textContent (ouverture dans un nouvel onglet)

    Etant donné que vous devez modifier plusieurs éléments en même temps, il faudra impérativement utiliser une boucle pour modifier la propriété .textContent sur chaque élément. Par exemple : 

    const listeElementsMaClasse = document.querySelectorAll(".ma-classe");
for (let i = 0; i < listeElementsMaClasse.length; i++) {
    listeElementsMaClasse[i].addEventListener('click', maFonction) 
}

Partie 5 (cliquer pour afficher / cacher)
  1. Toujours au clic sur le même bouton, ajoutez une classe CSS sur un seul élément. Il faudra utiliser un data-attribute sur l'élément que l'on souhaite cibler.

    Pour cibler par data-attribute, il faut mettre le nom du data-attribute entre crochets. Exemple :

    document.querySelector("[data-attribut-personnalise]")

    Le code ci-dessus cible donc le premier élément trouvé ayant le data-attribute "data-attribut-personnalise". A noter que cette syntaxe est aussi valable dans un fichier CSS.

    Si vous souhaitez cibler un data-attribut avec une valeur spécifique, il faut procéder de la façon suivante : 

    document.querySelector("[data-attribut-personnalise='ma valeur']");
    • Cette classe CSS, à ajouter/retirer, devra être rajoutée dans un fichier CSS. Pensez à mettre des propriétés CSS dans la classe pour voir des modifications ainsi que lier votre fichier html à votre fichier CSS.
    • Voir documentation la propriété classList (ouverture dans un nouvel onglet)
    • Voir documentation des data-attributes (ouverture dans un nouvel onglet)

Partie 6 (cliquer pour afficher / cacher)
  1. Ajoutez un champ de texte de type "text" (pas besoin de le mettre dans un formulaire)
  2. Ciblez le champ via un data-attribute et ajoutez l'évènement "input". Cet évènement permet d'écouter tout changement sur un champ "en temps réel"
  3. Affichez le contenu de la balise <input /> dans une balise <p> qui aura été elle-même ciblée avec un data-attribute

    Pour récupérer le texte écrit, il faudra récupérer l'argument de notre notion d'évènement. Par exemple :

    cible.addEventListener('input', (evt) => {
    // texteChamp contient le texte actuellement dans le champ de texte
    const texteChamp = evt.target.value;
})
    • Voir documentation l"évènement "input" (ouverture dans un nouvel onglet)
    • Voir documentation des data-attributes (ouverture dans un nouvel onglet)

Pour aller plus loin (cliquer pour afficher / cacher)
  • Ajoutez une balise <select> (pas besoin de la mettre dans un formulaire) avec une liste de choix que vous aurez choisi (n'oubliez pas la valeur de l'attribut "value" pour chaque "option"). Et une balise <img>. Lors de la sélection d'un élément dans le <select> l'image doit changer.

    N'oubliez pas de rajouter un data-attribute sur les balises <select> <img> pour pouvoir y accéder dans le javascript

    Il faudra télécharger des images (celles que vous voulez) et mettre le nom des images comme valeur de l'attribut "value". Quant à l'accès à l'attribut "src" de la balise <img> dans le javascript, il faut tout simplement accéder à la propriété "src" de la balise <img> dans le javascript (donc grâce à document.querySelector()).

    • Il faudra utiliser l'évènement "change" sur la balise <select>. Voir documentation l"évènement change (ouverture dans un nouvel onglet)
  • Ajouter plusieurs boutons de type "button" avec le même data-attribute mais des valeurs différentes. Arrangez-vous pour qu'au clic sur un des boutons (que vous avez ajouté), le libellé du bouton (propriété "textContent") change pour la valeur du data-attribute.

    Il faudra impérativement utiliser la propriété "target" de l'objet e, e est un paramètre de la fonction de la méthode addEventListener("nom-evenement", fonction)

    • Voir documentation la propriété textContent (ouverture dans un nouvel onglet)
    • Voir documentation la propriété (evt.)target (ouverture dans un nouvel onglet)
    • Voir documentation des data-attributes, elle montre comment récupérer la valeur d'un data-attribute (ouverture dans un nouvel onglet)
  • Faites une fonction qui retourne true ou false si une chaîne de caractères est en minuscules dans sa totalité.
