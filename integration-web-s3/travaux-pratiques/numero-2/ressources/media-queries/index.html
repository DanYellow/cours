Consignes de la pratique

Partie 1 (cliquer pour afficher / cacher)

Les règles @media à éditer sont déjà présentes à la fin du fichier "assets/css/style.css".

  1. Ouvrez le site contenu dans le dossier "exercice" (récupéré avec la ressource)
  2. Utilisez le mode responsive de votre navigateur (ou redimensionnez la fenêtre) pour voir comment le site s'adapte en fonction de la taille de l'écran (ou la fenêtre).
    • Accéder au mode responsive de Chrome/Opéra/Edge (ouverture dans un nouvel onglet)
    • Accéder au mode responsive de Firefox (ouverture dans un nouvel onglet)
    • Accéder au mode responsive de Safari (ouverture dans un nouvel onglet)
  3. Éditer la règle @media de façon à ce que le titre "Ma galerie d'images" soit plus gros sur un appareil dont la largeur d'écran est inférieure ou égale à 380px.

    Note : La règle @media est déjà présente vers la fin du fichier assets/css/style.css.

    • Voir documentation de la règle at-media (ouverture dans un nouvel onglet)
    • Voir documentation sur les media queries (ouverture dans un nouvel onglet)

    Pour rappel, si vous avez un Mac et un iPhone / iPad, vous pouvez consulter le site depuis votre terminal mobile en branchant votre appareil. Ceci fonctionne également avec le simulateur iOS. A condition que le site tourne sur un serveur.

    • Didacticiel pour inspecter un terminal réel sur macOS - Méthode 1
  4. Ajouter une règle @media pour gérer un écran dont la taille est supérieure ou égale à 400px et appliquer des déclarations CSS sur n'importe quel sélecteur CSS
  5. Éditer une règle @media pour gérer un écran ayant l'orientation paysage (landscape en anglais) et appliquer des déclarations CSS sur n'importe quel sélecteur CSS
  6. Ajouter / Editer le site pour le rendre responsive. Autrement dit rendre l'expérience agréable aussi bien sur PC, tablette et mobile
