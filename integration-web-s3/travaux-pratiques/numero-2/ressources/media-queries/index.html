Pré-consignes de la pratique - facultatives (cliquer pour afficher / cacher)

  1. Créez un nouveau fichier html (avec la structure de base d'un site web. Rappel, vous avez le raccourci "html:5")
  2. Copiez la balise <main> (et son contenu) dans le fichier HTML que vous venez de créer (si elle existe)
  3. Copiez les balises <link> présentes dans le <head> dans le fichier html que vous venez de créer
  4. Sauvegardez-le et commencez votre TP dedans

Consignes de la pratique

Partie 1 (cliquer pour afficher / cacher)

Les règles @media à éditer sont déjà présentes à la fin du fichier "assets/css/style.css".

  1. Utilisez le mode responsive de votre navigateur (ou redimensionnez la fenêtre) pour voir comment le site s'adapte en fonction de la taille.
    • Accéder au mode responsive de Chrome/Opéra/Edge (ouverture dans un nouvel onglet)
    • Accéder au mode responsive de Firefox (ouverture dans un nouvel onglet)
    • Accéder au mode responsive de Safari (ouverture dans un nouvel onglet)
  2. Éditer la règle @media de façon à ce que le titre "Ma galerie d'images" soit plus gros sur un appareil dont la largeur d'écran est inférieure ou égale à 380px.

    Note : La règle @media est déjà présente vers la fin du fichier assets/css/style.css

    • Voir documentation de la règle at-media (ouverture dans un nouvel onglet)
    • Voir documentation sur les media queries (ouverture dans un nouvel onglet)
  3. Ajouter une règle @media pour gérer un écran dont la taille est supérieure ou égale à 400px et appliquer des déclarations CSS sur n'importe quel sélecteur CSS
  4. Éditer une règle @media pour gérer un écran ayant l'orientation paysage (landscape en anglais) et appliquer des déclarations CSS sur n'importe quel sélecteur CSS
  5. Ajouter / Editer le site pour le rendre responsive. Autrement dit rendre l'expérience agréable aussi bien sur PC, tablette et mobile.

Ma galerie d'images

Tous les jours, une demi-heure de reportage sans commentaire. Inspirés par la célébrissime émission de radio américaine This American Life, "Les Pieds sur Terre" s’organisent désormais autour de récits, d’histoires vraies, une, deux ou trois par émission, qui tournent autour d’un même thème. Ces histoires sont racontées à la première personne et nourries d’éléments de reportage.
