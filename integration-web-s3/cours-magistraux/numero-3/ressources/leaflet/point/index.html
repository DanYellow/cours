CM - Leaflet

Leaflet est un plugin javascript open source permettant d'ajouter des données sur une carte OpenStreet Map. Tout comme le système de cartographie, l'outil est gratuit. Le but de cet exercice est de prendre en main. Bien évidemment nous allons respecter les bonnes règles pour notre utilisation de javascript en utilisant notamment un fichier javascript externe.
