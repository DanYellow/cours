Reverse tabnabbing

(cliquer pour afficher)

Le reverse tabnabbing est une faille de sécurité (corrigée sauf dans Internet Explorer) qui consiste à réécrire la page web ayant ouvert un nouvel onglet. Cette faille s'explique par la possibilité (en javascript) d'accéder à l'onglet (ou fenêtre) ayant ouvert le lien pointant vers une page malicieuse et la faire exécuter du javascript sans sa volonté. Ce code est souvent une redirection de page dans le but de faire du phishing (appelé aussi hameçonnage). Si les navigateurs modernes (sauf Internet Explorer, pour rappel) sont protégés par défaut, il est préférable de rajouter l'attribut "rel" avec la valeur "noopener" pour empêcher ce comportement.
