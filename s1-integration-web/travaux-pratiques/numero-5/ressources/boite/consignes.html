Modèle de boite et background — Consignes TP

Partie 1 (cliquer pour afficher / cacher)
  1. Ouvrez le fichier tp.html dans le navigateur. Qu'observez-vous ?

Partie 2 (cliquer pour afficher / cacher)
  1. Déplacez le contenu de la balise <style> du fichier tp.html dans un fichier CSS que vous aurez crée puis lié au fichier html tp.html
  2. Contrastez le texte du fichier tp.html grâce la propriété CSS "color" ou "background-color"
    • Accéder à la documentation de la propriété "background-color" (ouverture dans un nouvel onglet)
    • Accéder à la documentation de la propriété "color" (ouverture dans un nouvel onglet)

    N'oubliez pas de valider votre contraste avec la console du navigateur ou le site contrast.report (ouverture dans un nouvel onglet).

Partie 3 (cliquer pour afficher / cacher)
  1. Créez un paragraphe (balise <p>) avec dedans une balise <span>
  2. Ajoutez sur ces deux balises la même classe CSS
  3. Via une classe, rajoutez un background-color différent pour la balise <p> et <span>
  4. Dans le fichier CSS crée précedemment, ajoutez une nouvelle classe avec dedans les propriétés CSS "height" et "width"
    • Accéder à la documentation de la propriété "height" (ouverture dans un nouvel onglet)
    • Accéder à la documentation de la propriété "width" (ouverture dans un nouvel onglet)
  5. Qu'observez-vous ?

Partie 4 (cliquer pour afficher / cacher)
  1. Appliquez la propriété "padding" avec une valeur numérique et son unité sur la classe que vous avez crée précédemment
    • Accéder à la documentation de la propriété "padding" (ouverture dans un nouvel onglet)
    • Vous pouvez ajouter un background-color différent, via une classe dédiée, pour la balise <p> et le <span> pour mieux observez les changements
  2. Qu'observez-vous ?

Partie 5 (cliquer pour afficher / cacher)
  1. Créez une nouvelle balise de type bloc (<div>, <p>, <section>, ...) avec une nouvelle classe qui possède la propriété CSS margin. Les marges doivent être différentes pour chaque côté
    • Accéder à la documentation de la propriété "margin" (ouverture dans un nouvel onglet)

Partie 6 (cliquer pour afficher / cacher)
  1. Rajoutez une bordure sur une des classes que vous avez crée précédemment
    • Accéder à la documentation de la propriété "border" (ouverture dans un nouvel onglet)

Partie 7 (cliquer pour afficher / cacher)
  1. Appliquez la propriété outline sur une des classes que vous avez crée précédemment
    • Accéder à la documentation de la propriété "outline" (ouverture dans un nouvel onglet)

Partie 8 (cliquer pour afficher / cacher)
  1. Créez une nouvelle balise (de type bloc) avec une nouvelle classe. Cette classe doit répondre aux conditions suivantes :
    • Avoir une largeur fixe. Exemple : width: 420px
    • Avoir une bordure avec une épaisseur assez importante identique (au moins à gauche et à droite). Exemple : border: 6px solid #45BAFE
    • Avoir une marge intérieure (padding) fixe et identique (au moins à gauche et à droite). Exemple : padding: 20px
  2. Récupérez la largeur de cette nouvelle balise grâce à la console du navigateur. Quelle valeur pour la largeur de la balise obtenez-vous ?

    La largeur doit être récupérée en survolant la balise dans l'onglet "Elements/Inspector" ou en utilisant l'outil "flèche" (premier outil à gauche dans la console)

    Pour rappel, on affiche la console du navigateur grâce à la touche F12 ou clic droit > Inspecter / Examiner l'élément.

Partie 9 (cliquer pour afficher / cacher)
  1. Ajoutez le sélecteur suivant dans votre fichier CSS (de préférence au début du fichier) 
    *, *::before, *::after {
    box-sizing: border-box;
}
  2. Consultez une nouvelle fois la largeur de votre balise. Elle a maintenant la largeur que vous avez définie dans la propriété "width", et ce, malgré les padding et border appliqués
