Pour aller plus loin

Le service santé de CY Cergy Paris Université, en partenariat avec trois associations engagées, organise des distributions alimentaires gratuites sur trois sites différents. Cette initiative solidaire vise à soutenir les étudiants en leur offrant une aide dans leur quotidien.

L'attribut universel style contient des déclarations CSS afin de mettre en forme l'élément. Attention, il est recommandé de définir les règles de mise en forme dans un ou plusieurs fichiers séparés. Cet attribut, ainsi que l'élément <style>, ont simplement pour but de permettre une mise en forme rapide, notamment pour tester presto.

Cette méthode n'est pas vraiment recommandée, ainsi qu'utiliser la balise <style>. On l'a vu en cours mais on arrêtera après.
