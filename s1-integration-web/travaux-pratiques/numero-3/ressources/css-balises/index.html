CSS Balises — Consignes TP

Quand vous travaillez avec VS Code. Pensez toujours à glisser-déposer votre dossier de travail dans VS Code. Ceci vous permettra d'accéder très facilement à tous les fichiers de votre projet.

  1. Créez une nouvelle page HTML avec une balise <p>
  2. Ajouter l'attribut "style" à la balise <p> avec la déclaration CSS "color: red"
  3. Ajouter sur la même balise et dans le même attribut "style" la déclaration CSS "font-size: 2rem;"
  4. Ajouter une autre balise <p> avec la même valeur pour l'attribut "style" que la balise <p> précédemment écrite

Après chaque modification, n'oubliez pas de sauvegarder le fichier ( Ctrl/command (⌘) + s ) puis consultez votre page sur votre navigateur et pensez bien à actualiser votre page
