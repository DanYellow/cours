Flexbox — Consignes TP
  • Complétez le fichier index.html pour qu'il ressemble au plus proche au fichier "maquette-flexbox.jpg/.psd" fourni
    • Pour le texte en bas de page (après la grande image), il vous faudra également rajouter la structure HTML. Les textes ne sont pas soulignés pour rien, utilisez la balise appropriée
    • La grande image en bas de la maquette représente une vidéo youtube. A implémenter quand vous aurez fini la mise en page des parties du haut (voire partie "pour aller plus loin" pour plus d'infos)
    • Votre travail concernera principalement l'édition du fichier CSS ("ressources/css/style.css") à propos de flexbox.
      • Avant de développer. Pensez bien à utiliser la console du navigateur pour mieux comprendre le code fourni
      • Les images sont déjà dimensionnées correctement. Les problèmes se situent principalement au niveau de la structure de flexbox. Propriétés CSS manquantes

Le projet contient un fichier reset.css, le but de ce fichier est de réinitialiser les styles par défaut des navigateurs. Ainsi toutes les balises ont un style "nul". Par exemple, toutes les balises de texte ont la même taille.

Ce fichier de reset doit être le premier à être chargé dans votre fichier HTML. Ici, nous avons utilisé le reset d'Eric Meyer, c'est le plus populaire.

Pensez toujours à l'implémenter dans vos projets. A noter qu'il sera toujours présent dans les projets que nous vous remettrons.
