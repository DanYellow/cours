CSS Grid - Apple Store

Autre exemple de mise en page avec Flexbox. Ici des produits de la marque Apple sont affichés dans une grille dynamique permettant ainsi à chaque produit "d'affirmer" son identité. Là encore, nous avons utilisé les propriétés CSS grid-column-start/grid-column-end/grid-row-start/grid-row-end et la valeur span, apportant ainsi plus de souplesse en cas de modifications.

Autre avantage avec CSS Grid : il est très facile d'adapter la mise en page en fonction des écrans. Ceci sera vu en S3 lorsque nous aborderons le concept des media queries. Mais rien ne vous empêche de vous renseigner sur le web à propos de ce sujet dès aujourd'hui.
