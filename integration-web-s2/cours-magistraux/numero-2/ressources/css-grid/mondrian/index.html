CSS grid - Mondrian

Reproduire (plus ou moins) le tableau de Pete Mondrian "Composition avec grand plan rouge, jaune, noir, gris et bleu" est possible grâce à CSS grid. ça demande beaucoup de balises, mais l'utilisation reste simple.

Cet exemple est également l'occasion d'ouvrir la valeur "span" pour les propriétés CSS grid-column-start/grid-column-end/grid-row-start/grid-row-end. Cette valeur permet d'étendre un élément au lieu de définir une valeur de début et de fin. Exemple : 

.mon-selecteur {
    grid-column-start: 2;
    grid-column-end: span 3;
}

Dans l'exemple ci-dessus, on définit que notre élément commence à la colonne 2 et s'étendra de trois colonnes. L'avantage, c'est que si on change la valeur de "grid-column-start", l'élément occupera toujours trois colonnes.

Ce code équivaut à mettre la valeur "5" pour la propriété "grid-column-end".
