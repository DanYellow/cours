Consignes de la pratique

Partie 1 (cliquer pour afficher / cacher)
  1. Prenez en main le framework Tailwindcss en effectuant les tâches suivantes (un fichier tailwindcss.html est présent dans la ressource, vous pouvez travailler à partir de ce dernier) :
    • Un texte large et de couleur rouge (n'importe quel rouge)
    • Deux paragraphes de textes espacés par une marge basse (margin-bottom) pour les deux. Un des deux paragraphes doit contenir un lien en gras et de couleur verte
    • Une image contenue dans une balise. Cette balise doit avoir une marge intérieure (padding) et une bordure épaisse
    • Une balise d'une certaine taille (hauteur et largeur) avec une couleur d'arrière-plan (background-color)
    • Un paragraphe qui change de couleur de texte et de couleur d'arrière-plan au survol (:hover)
      • Accéder à la liste des modifiers (ouverture dans un nouvel onglet)
    • Une image avec du texte sur le côté. Texte à gauche ou à droite de l'image, c'est à votre convenance.
      • Il faudra impérativement utiliser flexbox
      • Pensez bien à espacer les éléments grâce à la propriété CSS gap
    • Voir documentation de tailwind (ouverture dans un nouvel onglet)
    • N'hésitez pas à utiliser la barre de recherche (recherche en anglais). Vous pouvez rechercher par propriétés CSS. Par exemple, pour la première tâche, vous pouvez rechercher "font-size" pour savoir comment gérer la taille du texte avec tailwindcss.

Sauf indications contraires, les tailles, teintes et couleurs pour les classes sont à votre discrétion.

    Vous trouverez également la maquette photoshop dans le dossier "img/" si ça peut vous aider. Mais encore une fois le but est d'être le plus proche et non d'être iso à 100% avec la maquette.

Les images n'ont pas à être celles de la maquette tout comme le texte (les images sont dans le dossier "/img"). Vous pouvez prendre des libertés (couleurs, tailles) à partir du moment où la mise en page est respectée.

Parallèlement, pour avoir les éléments côté-à-côte, il faudra utiliser flexbox. En savoir plus sur Flexbox dans tailwind.

Pour l'arrière-plan (le background), c'est un dégradé. Vous pouvez le générer en ligne avec https://www.creative-tim.com/twcomponents/gradient-generator - Ouverture dans un nouvel onglet
