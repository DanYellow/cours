Consignes de la pratique

Partie 1 (cliquer pour afficher / cacher) - Position relative

  1. Créez une nouvelle page HTML avec son fichier CSS associé
  2. Créez une balise et mettez-la en position relative grâce au CSS
  3. Utilisez les propriétés "top", "bottom", "left" ou "right" pour déplacer la balise
    • Voir documentation de la propriété position (ouverture dans un nouvel onglet)
    • Voir documentation de la propriété top (ouverture dans un nouvel onglet)
    • Voir documentation de la propriété bottom (ouverture dans un nouvel onglet)
    • Voir documentation de la propriété left (ouverture dans un nouvel onglet)
    • Voir documentation de la propriété right (ouverture dans un nouvel onglet)

Partie 2 (cliquer pour afficher / cacher) - Position absolute

  1. Créez une balise et mettez-la en position absolute grâce au CSS
  2. Utilisez les propriétés "top", "bottom", "left" ou "right" pour déplacer la balise
    • Voir documentation de la propriété position (ouverture dans un nouvel onglet)
    • Voir documentation de la propriété top (ouverture dans un nouvel onglet)
    • Voir documentation de la propriété bottom (ouverture dans un nouvel onglet)
    • Voir documentation de la propriété left (ouverture dans un nouvel onglet)
    • Voir documentation de la propriété right (ouverture dans un nouvel onglet)

Partie 3 (cliquer pour afficher / cacher) - Position fixed

  1. Placez en "position: fixed" une balise avec une classe CSS
  2. Déplacez cette balise grâce aux propriétés "top", "bottom", "left" ou "right"
  3. Faites défiler votre page, la balise devrait être toujours visible
    • Voir documentation de la propriété position (ouverture dans un nouvel onglet)
    • Voir documentation de la propriété top (ouverture dans un nouvel onglet)
    • Voir documentation de la propriété bottom (ouverture dans un nouvel onglet)
    • Voir documentation de la propriété left (ouverture dans un nouvel onglet)
    • Voir documentation de la propriété right (ouverture dans un nouvel onglet)

    Si votre page n'est pas assez haute, n'hésitez pas à modifier la hauteur de la balise lt;body> (propriété CSS "height")

Partie 4 (cliquer pour afficher / cacher) - Position sticky

  1. Ajoutez une nouvelle balise en position "sticky"
  2. Avec les propriétés "top", "bottom", "left" ou "right" déplacez la balise

    Si votre balise en position "sticky" n'a pas été déplacée grâce aux propriétés CSS "top", "left", "right" ou "bottom". La valeur "sticky" ne sera pas appliquée.

  3. Défilez sur la page normalement, une fois la balise atteinte elle restera visible mais reprendra sa place lorsque vous serez à son positionnement habituel (celui dans le flux)
    • Voir documentation de la propriété position (ouverture dans un nouvel onglet)
    • Voir documentation de la propriété top (ouverture dans un nouvel onglet)
    • Voir documentation de la propriété bottom (ouverture dans un nouvel onglet)
    • Voir documentation de la propriété left (ouverture dans un nouvel onglet)
    • Voir documentation de la propriété right (ouverture dans un nouvel onglet)

Partie 5 (cliquer pour afficher / cacher) - z-index

  1. Utilisez la propriété CSS "z-index" pour déplacer vos balises sur l'axe z
  2. Utilisez la propriété "z-index" pour changer le chevauchement des balises dont le positionnement est différent de "static"
    • Voir documentation de la propriété z-index (ouverture dans un nouvel onglet)
