Pré-consignes de la pratique - facultatives (cliquer pour afficher / cacher)

  1. Créez un nouveau fichier html (avec la structure de base d'un site web. Rappel, vous avez le raccourci "html:5")
  2. Copiez la balise <main> (et son contenu) dans le fichier HTML que vous venez de créer (si elle existe)
  3. Copiez les balises <link> présentes dans le <head> dans le fichier html que vous venez de créer
  4. Sauvegardez-le et commencez votre TP dedans

Consignes de la pratique

Partie 1 (cliquer pour afficher / cacher)
  1. Appliquez une transformation (de votre choix sur l'élément - classe "conteneur" - présent sur la page)
    • Voir documentation de la propriété transform (ouverture dans un nouvel onglet)
    • Voir listes des fonctions de transformation transform (ouverture dans un nouvel onglet)
  2. Utilisez la propriété CSS "transition" pour appliquer la transformation effectuée à l'étape précédente au survol
    • Voir documentation de la propriété transition (ouverture dans un nouvel onglet)

    Pour rappel, la propriété CSS "transition" doit être mise sur l'état de départ, pas dans le ":hover".

Partie 2 (cliquer pour afficher / cacher)
  1. Complétez le sélecteur ".conteneur::after" de façon à rajouter un arrière-plan (aplat de couleur ou dégradé) permettant de faire ressortir le texte non-visible
    • Voir documentation du pseudo-élément "after" transition (ouverture dans un nouvel onglet)
    • Voir documentation du pseudo-élément "before" (ouverture dans un nouvel onglet)
    • Voir documentation du pseudo-élément "background-color" (ouverture dans un nouvel onglet)
    • Générateur de dégradé (ouverture dans un nouvel onglet)
    • Rappel : Pour étirer une balise en position absolute (ou relative), aidez-vous du code ci-dessous. 
                                      mon-sélecteur {
    position: absolute;
    top: 0;
    left: 0;
    right: 0;
    bottom: 0;
    /* propriétés CSS */
}
  2. Utilisez la pseudo-classe "hover" pour afficher le background au survol l'arrière-plan précédemment affiché
    • Au lieu d'utiliser la propriété CSS "transform" vous pouvez utiliser la propriété CSS "opacity"
      • Voir documentation du pseudo-élément "opacity" (ouverture dans un nouvel onglet)
      Bien évidemment, il faudra penser à cacher par défaut l'arrière-plan pour l'afficher ensuite

Partie 3 (cliquer pour afficher / cacher)
  1. Utilisez la propriété CSS transform et la fonction translate() (translateX() ou translateY() ) pour déplacer les textes présents sur l'image lors du survol de la base .conteneur.
    • Vous pouvez déplacer les deux textes ensemble ou un seul à la fois à vous de voir
    • Utilisez la console du navigateur pour trouver les sélecteurs
    • Voir documentation de la propriété transform (ouverture dans un nouvel onglet)

Partie 4 (cliquer pour afficher / cacher)

Maintenant que vous avez expérimenté de façon guidée les transitions et transformation à vous de jouer.

  1. Au choix :
    • Copiez la balise "conteneur-principal" et son contenu et collez-la après la première balise "conteneur-principal"
    • Ajoutez votre propre balise avec quelques éléments enfants (images, liens, textes...)
  2. Appliquez une ou plusieurs transitions de votre choix au survol
