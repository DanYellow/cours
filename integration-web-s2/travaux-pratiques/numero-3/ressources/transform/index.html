Consignes de la pratique

Partie 1 (cliquer pour afficher / cacher)
  1. Ouvrez le site "exercice/" (récupéré avec la ressource)
  2. Appliquez une transformation de votre choix sur l'élément ayant la classe "conteneur". Cet élément est présent dans le site précédemment ouvert
    • Voir documentation de la propriété transform (ouverture dans un nouvel onglet)
    • Voir listes des fonctions de la propriété "transform" (ouverture dans un nouvel onglet)
  3. Utilisez la propriété CSS "transition" pour appliquer la transformation effectuée à l'étape précédente au survol ou en utilisant la règle-at @starting-style
    • Voir documentation de la propriété transition (ouverture dans un nouvel onglet)

    Pour rappel, la propriété CSS "transition" doit être mise sur l'état de départ, pas dans la pseudo-classe.

Partie 2 (cliquer pour afficher / cacher)
  1. Complétez le sélecteur ".conteneur::after", du fichier CSS fourni, de façon à rajouter un arrière-plan (aplat de couleur, dégradé ou même image) permettant de faire ressortir le texte au faible contraste en effectuant une transition
    • Voir documentation du pseudo-élément "after" transition (ouverture dans un nouvel onglet)
    • Voir documentation du pseudo-élément "before" (ouverture dans un nouvel onglet)
    • Voir documentation du pseudo-élément "background-color" (ouverture dans un nouvel onglet)
    • Générateur de dégradé (ouverture dans un nouvel onglet)

    Pour rappel, pour étirer une balise en position absolute (ou fixed), aidez-vous du code ci-dessous :

    • .mon-sélecteur {
    position: absolute;
    top: 0;
    left: 0;
    right: 0;
    bottom: 0;
    /* [...] */
}
    • .mon-sélecteur {
    position: absolute;
    inset: 0;
    /* [...] */
}

    Les deux codes ci-dessus font la même chose.

  2. Utilisez la pseudo-classe "hover" pour changer la couleur de la propriété CSS "background" du pseudo-element ajouté précédemment affiché

Partie 3 (cliquer pour afficher / cacher)
  1. Utilisez la propriété CSS transform et la fonction translate() (translateX() ou translateY()) pour déplacer les textes présents sur l'image lors du survol de la balise ".conteneur".
    • Vous pouvez déplacer les deux textes ensemble ou un seul à la fois, à vous de voir
    • Voir documentation de la propriété transform (ouverture dans un nouvel onglet)

Partie 4 (cliquer pour afficher / cacher)

Maintenant que vous avez expérimenté de façon guidée les transitions CSS à vous de jouer.

  1. Au choix :
    • Réutilisez la balise "conteneur-principal" et son contenu
    • Ajoutez votre propre balise avec quelques éléments enfants (images, liens, textes...)
  2. Appliquez une ou plusieurs transitions de votre choix au survol (et au focus clavier si éligible)
