Consignes de la pratique - Transition CSS

Partie 1 (cliquer pour afficher / cacher)
  1. Créez un nouveau fichier HTML et liez-le à un fichier CSS
  2. Ajoutez un lien (balise <a>) avec une classe et la pseudo-classe ":hover" qui permet de changer la couleur du texte au survol
    • Documentation de la pseudo-classe :hover - Ouverture dans un nouvel onglet

Partie 2 (cliquer pour afficher / cacher)
  1. Appliquez une transition sur le lien précédemment crée
    • Voir documentation de la propriété transition - Ouverture dans un nouvel onglet
    • Voir documentation de la propriété color - Ouverture dans un nouvel onglet

    On l'a vu en cours, il est possible de jouer sur l'interpolation de la transition via la propriété CSS "transition-timing-function". Plusieurs valeurs sont possibles et vous pouvez créer votre propre courbe (notamment grâce à la console du navigateur). Si vous souhaitez avoir un affichage visuel de ces fonctions, vous avez le site suivant : https://www.joshwcomeau.com/animation/css-transitions/#timing-functions-2 (ouverture dans un nouvel onglet)

  2. Survolez le lien, que constatez-vous ?
    • Une fois cette consigne effectuée, essayez de placer la propriété CSS "transition" (et sa valeur) dans la pseudo-classe ":hover" de la classe de votre lien

Pour des questions d'accessibilité, il faut toujours permettre aux utilisateurs, qui le souhaitent, de désactiver les animations, notamment les transitions CSS. Pour ce faire, il vous suffit d'ajouter le code suivant :

@media (prefers-reduced-motion: reduce) {
    *,
    ::before,
    ::after {
        animation-delay: -1ms !important;
        animation-duration: 1ms !important;
        animation-iteration-count: 1 !important;
        background-attachment: initial !important;
        scroll-behavior: auto !important;
        transition-duration: 0s !important;
        transition-delay: 0s !important;
        view-transition-name: unset !important;
    }

    ::view-transition-group(*),
    ::view-transition-old(*),
    ::view-transition-new(*) {
        animation: none !important;
    }
}

Pensez toujours à mettre ceci dans votre code, ça ne coûte pas grand-chose et ceci améliorera l'accessibilité de votre site. Ce bout de code sera toujours mis dans les projets que nous vous fournirons.

Partie 3 (cliquer pour afficher / cacher)
  1. Effectuez une transition sur votre lien sur la propriété "margin-top", "margin-bottom" ou "margin-left"
    • Voir documentation de la propriété margin-left - Ouverture dans un nouvel onglet
    • Voir documentation de la propriété margin-top - Ouverture dans un nouvel onglet
    • Voir documentation de la propriété margin-bottom - Ouverture dans un nouvel onglet

    Les propriétés "margin-top" et "margin-bottom" ne fonctionneront pas si votre balise <a> est toujours du type "inline".

Partie 4 (cliquer pour afficher / cacher)
  1. Ajoutez une balise qui contient d'autres balises
  2. Définissez un nouveau sélecteur CSS permettant d'effectuer une transition sur les éléments contenus dans la balise en même temps
    • Ce nouveau sélecteur CSS doit impérativement impliquer le nom de la classe du conteneur, il vous faudra donc jouer sur l'imbrication de sélecteurs CSS (voir exemple dans la présentation du cours associé)

    Chaque élément peut avoir une transition différente

  3. Survolez la balise contenant le tout pour valider votre code

Partie 5 (cliquer pour afficher / cacher)
  1. Utilisez la règle-at @starting-style pour exécuter une transition lors du chargement de la page
    • Voir documentation de la règle-at @starting-style - anglais - Ouverture dans un nouvel onglet

Si vous souhaitez observer l'application de la règle-at @starting-style, il faudra penser à recharger la page.
