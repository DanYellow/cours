Consignes de la pratique

Partie 1 (cliquer pour afficher / cacher)
  1. Créez un nouveau fichier HTML et liez-le à un fichier CSS
  2. Ajoutez un lien (balise <a>) avec une classe et la pseudo-classe ":hover" qui permet de changer la couleur du texte au survol

Partie 2 (cliquer pour afficher / cacher)
  1. Appliquez une transition sur le lien précédemment crée
    • Voir documentation de la propriété transition (ouverture dans un nouvel onglet)
    • Voir documentation de la propriété color (ouverture dans un nouvel onglet)
  2. Survolez le lien, que constatez-vous ?
    • Une fois cette consigne effectuée, essayez de placer la propriété CSS "transition" (et sa valeur) dans la pseudo-classe ":hover" de la classe de votre lien

Partie 3 (cliquer pour afficher / cacher)
  1. Effectuez une transition sur votre lien sur la propriété "margin-top", "margin-bottom" ou "margin-left"
    • Voir documentation de la propriété margin-left (ouverture dans un nouvel onglet)
    • Voir documentation de la propriété margin-top (ouverture dans un nouvel onglet)
    • Voir documentation de la propriété margin-bottom (ouverture dans un nouvel onglet)

Partie 4 (cliquer pour afficher / cacher)
  1. Ajoutez un paragraphe et mettez-le avec la balise <a> crée précédemment dans une balise qui contiendra les deux balises
  2. Ajoutez un nouveau sélecteur CSS permettant d'effectuer une transition sur les deux éléments contenus dans la balise en même temps
    • Ce nouveau sélecteur CSS doit impérativement impliquer le nom de la classe du conteneur, il vous faudra donc jouer sur l'imbrication de sélecteurs CSS (voir exemple dans la présentation du cours associé)
  3. Survolez la balise contenant le tout pour valider votre code

Partie 5 (cliquer pour afficher / cacher)
  1. Ajoutez une nouvelle balise contenant du texte, des images ou autres (ou même les trois)
  2. Ajoutez une transition pour permettre de jouer sur l'opacité de la balise ajoutée précédemment
    • Voir documentation du pseudo-élément "opacity" (ouverture dans un nouvel onglet)
  3. Survolez votre nouvelle balise, elle doit s'afficher. Que constatez-vous ?

Pour aller plus loin

  • Refaites cet exercice avec tailwind css (ouverture dans un nouvel onglet)
    • Cliquez sur le bouton "Get Started"
    • Cliquez sur l'onglet "Play CDN"
    • Copiez le code de la partie 1 dans votre fichier HTML. Et voilà, tailwindcss est prêt
    • Consultez la documentation et/ou aidez vous du moteur de recherche pour trouver la classe CSS correspondant à vos besoins
