Balise <template> — Consignes TP

Partie 1 (cliquer pour afficher / cacher)
  1. En vous inspirant de l'exemple dans les slides du cours, créez et affichez un template HTML sur votre page.
    • Si vous avez oublié "html:5" puis touche entrée, vous crée un squelette de page html
    • Voir documentation de <template> - ouverture dans un nouvel onglet

N'oubliez pas de créer un fichier javascript pour charger puis afficher votre template. Et bien évidemment de cloner votre template avant de l'utiliser.

Partie 2 (cliquer pour afficher / cacher)
  1. Arrangez-vous pour modifier le contenu d'une ou plusieurs balises de votre template depuis le javascript.
    • Cette modification peut être de différent ordre : texte, lien d'image... Petit rappel sur les propriétés JS pour modifier un élément HTML :
      • textContent - Texte
      • setAttribute - Gestion des attributs
      • classList - Modification des classes CSS

S'il est préférable d'utiliser un data-attribute pour cibler un élément. Vous pouvez aussi cibler directement la balise si la structure est simple.

Partie 3 (cliquer pour afficher / cacher)
  1. Ajoutez un bouton qui, au clic, effectue les actions suivantes :
    • Appelle la fonction "getListNationalHolidays" (contenue dans le fichier index.js)
    • Affiche les dates grâce à la balise <template>. Ceci peut être via une liste (<ol>) ou un tableau (<table>)

L'API retourne une liste d'objets ayant la structure : {name, value}

Pour aller plus loin (cliquer pour afficher / cacher)
    • Groupez les jours fériés par année
    • Affichez le contenu de chaque année avec les balises <details> et <summary>
    • Proposez à l'utilisateur de sélectionner les différentes régions et mettez à jour les informations en fonction

Pour proposer de meilleures performances, vous pouvez mettre les données en cache pour éviter un appel d'api si vous avez déjà chargé ces données. La liste des régions est trouvable dans la documentation.
