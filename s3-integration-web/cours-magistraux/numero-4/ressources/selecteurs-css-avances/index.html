Cours Magistral - Leaflet

Ce TP est composé de deux parties concernant l'utilisation de Leaflet. Pour rappel, Leaflet est un plugin javascript open source et gratuit permettant d'ajouter des données sur une carte OpenStreetMap. Tout comme le système de cartographie, Leaflet est gratuit. Le but de cette ressource est de vous montrer divers exemples concernant l'outil.

Veillez à vérifier le code source, il est commenté. Et bien évidemment n'hésitez pas à effectuer des modifications pour mieux comprendre comment ça fonctionne.
