Prenez en compte les notes suivantes :

  • Le site de base est responsive et utilise le CSS nesting pour "économiser" des lignes code. Si le sujet n'a pas encore été abordé en cours, vous trouverez ci-dessous une documentation pour en savoir plus. En tous les cas, son utilisation reste facultative
    • Voir documentation sur le CSS Nesting
  • Le rose utilisé dans les lignes du tableau est le #fce7ec
  • La police utilisée pour les titres est la Noto Sans
  • Des libertés ont été prises comparé à la maquette, au niveau des couleurs et des textes. En revanche, l'affichage ou non des éléments, indiqué dans la maquette, doit être respecté grâce aux sélecteurs CSS et pseudo-classes
  • La mise en page du site, au niveau des pâtisseries, est faite avec CSS Grid. Un TP abordera également ce sujet lors de ce semestre
    • Voir documentation de CSS Grid
  1. Utilisez les sélecteurs avancés vus en cours pour améliorer le code fourni pour qu'il colle le plus possible à la maquette fournie

    Vous ne devez pas ajouter de nouvelles classes CSS, seuls les sélecteurs avancés et pseudo-classes sont autorisés.

    • Voir documentation des sélecteurs CSS
    • Voir documentation des pseudo-classes
  2. Ajoutez des transitions CSS sur les éléments qui changent d'état au survol / focus clavier
    • Voir documentation des transitions CSS
