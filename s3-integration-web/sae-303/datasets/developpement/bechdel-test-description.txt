- bt_score: le score de Bechdel. Nombre entier allant de 0 à 3
    - 0 : Il n'y a qu'une seule femme (ou moins)
    - 1 : Les femmes ne parlent pas
    - 2 : Les femmes ne parlent que d'hommes
    - 3 : Le film réussit le test
- imdbid: The IMDb id
- title: The title of the movie. Any weird characters are HTML encoded (so Brüno is returned as "Br&uuml;no").
- year:	L'année de sortie du film (selon le site web IMDb)
- title_fr: titre en français du film
- imdbid_full : ID imdb du film complet
- poster_path : affiche française du film
- lang : langue d'origine du film - Vous pourrez ainsi filtrer pour garder des films de certains pays

Pour plus d'informations : https://bechdeltest.com/api/v1/doc
