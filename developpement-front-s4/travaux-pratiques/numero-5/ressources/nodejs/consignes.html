Consignes de la pratique

Partie 1 (cliquer pour afficher / cacher)
  1. Installez nodejs sur votre ordinateur
    • Installez nodejs (ouverture dans un nouvel onglet) - Prenez la version LTS, plus stable
  2. Ouvrez le logiciel cmd / invite de commandes (Windows) ou terminal (MacOS)
  3. Entrez la commande "node" puis tapez entrée. Si l'installation de node s'est bien passée, vous devriez voir le numéro de version
    • Appuyez sur les touches ( Ctrl/command (⌘) + c ) deux fois pour sortir de l'invite de commande nodejs
  4. Ouvrez votre dossier de travail, celui où vous comptez faire votre TP

    Il est très fortement conseillé d'installer le logiciel gratuit "cmder", il permet d'avoir une console sous Windows avancée et permettant d'utiliser les commandes Unix sous Windows ( cd , mkdir , echo ...)

    • Télécharger cmder (prendre la version full) (Ouverture dans un nouvel onglet)
    1. Ecrivez "cd" suivi d'un espace dans le logiciel invite de commandes / cmd
    2. Glissez le dossier de travail
    3. Appuyez sur la touche Entrée

    Il est très fortement conseillé d'installer le logiciel gratuit "iterm2", il permet de remplacer la Terminal natif de MacOS qui est très limité

    • Télécharger iTerm2 (Ouverture dans un nouvel onglet)
    1. Ecrivez "cd" suivi d'un espace dans le logiciel Terminal
    2. Glissez le dossier de travail
    3. Appuyez sur la touche Entrée
  5. Vous devriez voir dans le terminal (ou cmd) le chemin de votre dossier suivi d'un ">"
    • Par exemple : E:\mes_projets\tps\developpement_front\nodejs>

Partie 2 (cliquer pour afficher / cacher)

Assurez-vous bien d'être dans votre dossier de travail dans le terminal !

  1. Entrez la commande npm init -y
  2. Installez le node_modules "lodash"
    • Rappel, l'installation se fait grâce à la ligne de commande "npm install _nom_du_module_" dans le terminal
      • npm install lodash
    • Voir la fiche npm de lodash (ouverture dans un nouvel onglet)
  3. Qu'observez-vous dans le fichier package.json ?

Partie 3 (cliquer pour afficher / cacher)
  1. Créez un fichier "index.js"
  2. Importez-y lodash
    • import _ from "lodash";
  3. Appliquez une fonction de lodash dans votre fichier "index.js" et affichez le résultat dans un console.log()
    • Voir documentation de lodash
  4. Modifiez le fichier package.json de façon à ajouter une tâche "start" qui va exécuter votre fichier "index.js"
    • Pour rappel, les tâches doivent être mises dans la partie "scripts" du fichier package.json
  5. Rajoutez "type": "module" à la racine du fichier package.json
    • Ceci va nous permettre d'utiliser la syntaxe import x from 'x' côté serveur
  6. Entrez la commande npm run start dans le terminal et appuyez sur la touche entrée pour afficher le contenu du console.log()

Partie 4 (cliquer pour afficher / cacher)
  1. Créez un module javascript et exportez-le
    • Pour rappel, vous ne pouvez avoir qu'un seul export default par module
    • Votre module peut être à peu près ce que vous voulez (fonction, chaîne de caractères...)
  2. Importez et utilisez votre module exporté dans index.js
  3. Observez le résultat dans le terminal grâce à la commande npm run start
