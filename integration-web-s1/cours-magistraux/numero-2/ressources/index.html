Cours Magistral S1 - Référencement Naturel / Search Engine Optimization

Le SEO (Search Engine Optimization) ou Référencement Naturel représente une part importante dans la construction d'un site web à quoi bon de développer le meilleur site du monde si à la fin personne ne le trouve.

Dans cette ressource, vous avez deux versions du même site, si l'apparence est la même, la structure du code HTML est, quant à elle, différente. Les balises sont mieux choisies ainsi que les attributs. On l'a vu en cours, le code HTML représente une grande part du référencement d'un site web, un code de mauvaise qualité peut très fortement sanctionner un site web.

Grâce à lighthouse, vous pouvez tester le score de ces deux sites. Attention tout de même, Lighthouse ne fonctionne qu'avec le protocole HTTP(S), il faut donc que le site tourne sur un serveur. De même pour Screamingfrog. Pensez à déplacer le contenu des sites dans WAMP/MAMP ou un serveur distant.
