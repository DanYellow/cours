Formattage-texte — Consignes TP (cliquer pour afficher / cacher)

Pensez bien à utiliser une classe CSS dédiée pour chaque étape de la consigne.

N'oubliez pas qu'il n'est pas possible d'imbriquer une balise <p> dans une autre balise <p>. Pour styliser du texte dans un paragraphe (balise <p>) il faut utiliser une balise <span>.

  1. Créez un nouveau fichier html et un fichier css. Et liez les deux fichiers grâce à la balise <link>
    • Voir documentation balise <link> (ouverture dans un nouvel onglet)
  2. Ajoutez des paragraphes (balise <p>) puis en utilisant une classe CSS dédiée, changez l'alignement des paragraphes (différent pour chaque paragraphe)
    • Voir documentation propriété text-align (ouverture dans un nouvel onglet)
  3. Changez la police d'écriture (font-family) d'une partie du texte

    Ne pas oublier les polices de secours.

    • Voir documentation propriété font-family (ouverture dans un nouvel onglet)
  4. Au sein d'un même paragraphe : Souligner et rendre gras une partie du texte
    • Voir documentation propriété font-weight (ouverture dans un nouvel onglet)
    • Voir documentation propriété text-decoration (ouverture dans un nouvel onglet)
  5. Au sein d'un même paragraphe : Augmentez la taille d'une partie du texte et pour une autre partie du texte : Diminuez la taille du texte
    • Voir documentation propriété font-size (ouverture dans un nouvel onglet)

    N'oubliez pas d'utiliser l'unité rem et non px.

    • En savoir plus sur l'unité rem - Ouverture dans un nouvel onglet
    Conversion px → rem rem = Taille en pixels 16
  6. Pour une partie de texte : Ajoutez une couleur d'arrière-plan
    • Voir documentation propriété background-color (ouverture dans un nouvel onglet)
  7. Créez une liste ordonnée et une autre non-ordonnée
    • Voir documentation balise <ol> (ouverture dans un nouvel onglet)
    • Voir documentation balise <ul> (ouverture dans un nouvel onglet)
    • Voir documentation balise <li> (ouverture dans un nouvel onglet)
