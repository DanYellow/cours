Travaux pratiques sur les images responsives

But de la pratique : Voir la balise <picture>

Balise <picture>

Consigne : Compléter le code suivant de façon à ce que

  • L'image assets/images/balise-picture/salameche.png s'affiche pour un écran dont la taille est inférieure ou égale à 767px (déjà présent dans le code)
  • L'image assets/images/balise-picture/reptincel.png s'affiche pour un écran dont la taille est comprise entre 768px et 1024px
  • L'image assets/images/balise-picture/dracaufeu.png s'affiche pour un écran dont la taille est comprise entre 1025px et 1099px
  • L'image assets/images/balise-picture/mega-dracaufeu-x.png s'affiche pour un écran dont la densité de pixel est égale à 2
  • L'image assets/images/balise-picture/mega-dracaufeu-y.png s'affiche pour un écran dont la densité de pixel est égale à 3
  • L'image assets/images/balise-picture/defaut.png s'affiche lorsqu'aucune des condititions suscitées n'est appliquable.
