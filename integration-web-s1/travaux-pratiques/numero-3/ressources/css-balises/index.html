Consignes de la pratique
  1. Créez une nouvelle page html avec une balise <p>
  2. Ajouter l'attribut "style" à la balise <p> avec la valeur "color: red"
  3. Ajouter sur la même balise et dans la valeur de l'attribut "style", la déclaration "font-size: 2rem;"
  4. Ajouter une balise <p> avec la même valeur pour l'attribut "style" que la balise <p> précédemment écrite

Après chaque modification, n'oubliez pas de sauvegarder le fichier ( Ctrl/command (⌘) s ) puis consultez votre page sur votre navigateur et pensez bien à actualiser votre page
