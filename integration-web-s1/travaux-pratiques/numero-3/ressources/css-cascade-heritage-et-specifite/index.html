Pré-consignes de la pratique - facultatives (cliquer pour afficher / cacher)

  1. Créez un nouveau fichier html (avec la structure de base d'un site web. Rappel, vous avez le raccourci "html:5") dans le même dossier que celui-ci.
  2. Copiez les balises contenues entre les commentaires "<!-- Début: [...] -->" et "<!-- Fin: [...] -->"
    Il peut avoir plusieurs commentaires du même genre au sein du même fichier.
  3. Collez leur contenu dans votre même fichier aux mêmes endroits. Par exemple, si vous copiez du code dans le balise <head>, il faudra le coller également dans la balise <head> dans votre fichier.
  4. Sauvegardez-le et commencez votre TP dedans.

Consignes de la pratique - Cascade

(cliquer pour afficher)
  • Voir documentation de la cascade
  1. A la fin du fichier "style.css", ajoutez des classes identiques à celles déjà présentent dans le fichier
  2. Ajoutez aux classes que vous venez d'ajouter avec les mêmes déclarations CSS (que celles déjà présentent dans le fichier) avec des valeurs différentes
  3. Sauvegardez et regardez dans votre navigateur
    • Qu'observez-vous ?
  4. Déplacez les classes que vous avez ajouté (et leur contenu) au début du fichier "style.css" (donc au-dessus des classes initialement présentes dans le fichier)
  5. Sauvegardez et regardez dans votre navigateur
    • Qu'observez-vous ?

Consignes de la pratique - Héritage

(cliquer pour afficher)
  • Voir documentation de la héritage
  1. Ajoutez la déclaration CSS "font-weight: bold" sur la classe "liens-utiles"
    • Ajoutez le sélecteur pour la classe "liens-utiles" si elle n'est pas présente dans le fichier "style.css"
  2. Sauvegardez et regardez dans le navigateur
    • Qu'observez-vous ?

Consignes de la pratique - Spécificité

(cliquer pour afficher)
  • Voir documentation de la héritage
  1. Appliquez des déclarations CSS à tout élément ayant la classe "titre-principal" contenu dans un élément ayant la classe "en-tete"
    • Liste de propriétés CSS que vous pouvez appliquer :
      • background-color: Voir documentation de la propriété background-color (ouverture dans un nouvel onglet)
      • padding: Voir documentation de la propriété padding (ouverture dans un nouvel onglet)
      • margin: Voir documentation de la propriété margin (ouverture dans un nouvel onglet)
      • Vous pouvez bien évidemment utiliser celles qu'on a vu auparavant comme : font-size, color, text-decoration...
    • Note : le contenu de votre sélecteur ne doit en aucun cas affecter un élément ayant la classe "titre-principal" contenu dans un élément ayant la classe "nouvelle"
  2. Appliquez des déclarations CSS à tout élément ayant la classe "titre-principal" contenu dans un élément ayant la classe "nouvelle"
    • Voir plus haut pour voir la liste des propriétés CSS utilisables
    • Note : le contenu de votre sélecteur ne doit en aucun cas affecter un élément ayant la classe "titre-principal" contenu dans un élément ayant la classe "en-tete"
  3. En utilisant des classes, appliquez une couleur spécifique pour chaque lien wikipédia décrivant le contenu d'un BUT.
    • Note : La classe utilisée doit être la même pour chaque lien, il faudra obligatoirement jouer sur le sélecteur descendant (donc la spécificité) pour réaliser cette tâche

Je suis le texte exemple du TP

Cet exercice vise à découvrir les notions de spécification, cascade et héritage du CSS. Ces notions sont clés pour la compréhension de la matière, ce n'est pas à apprendre par coeur, mais à mettre dans un coin de sa tête pour debbuger ses problèmes de CSS.
