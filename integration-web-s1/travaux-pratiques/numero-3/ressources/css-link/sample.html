Pré-consignes de la pratique - facultatives (cliquer pour afficher / cacher)

  1. Créez un nouveau fichier html (avec la structure de base d'un site web. Rappel, vous avez le raccourci "html:5") dans le même dossier que celui-ci.
  2. Copiez les balises contenues entre les commentaires "<!-- Début: [...] -->" et "<!-- Fin: [...] -->"
    Il peut avoir plusieurs commentaires du même genre au sein du même fichier.
  3. Collez leur contenu dans votre fichier aux mêmes endroits. Par exemple, si vous copiez du code dans le balise <head>, il faudra le coller également dans la balise <head> dans votre fichier.
  4. Sauvegardez-le et commencez votre TP dedans.

Consignes de la pratique (cliquer pour afficher / cacher)

  1. Créez un fichier .css au même niveau que le fichier index.html appelé style.css (ou un autre nom)
  2. Liez le fichier .css à ce fichier index.html, grâce à la balise <link> (voir dans le <head> pour avoir un exemple)
  3. Coupez-collez le contenu de la balise <style> dans le fichier style.css (que vous venez de créer) puis supprimez la balise vidée de son contenu
  4. Ecrivez le contenu des attributs "style" dans le fichier style.css (que vous venez de créer) avec les sélecteurs de classe associés (vous devez impérativement créer des classes CSS si la balise n'en possède pas et lui associer la classe)
  5. Supprimez ensuite les attributs "style" et leurs valeurs
  6. Ajoutez une nouvelle balise, associez-lui une des classes que vous avez crée précédemment
  7. Confirmez la bonne réalisation de l'opération en consultant la page sur votre navigateur

L'attribut universel style contient des déclarations CSS afin de mettre en forme l'élément. Attention, il est recommandé de définir les règles de mise en forme dans un ou plusieurs fichiers séparés. Cet attribut, ainsi que l'élément <style> ont simplement pour but de permettre une mise en forme rapide, notamment pour tester.

Cette méthode n'est pas vraiment recommandée, ainsi qu'utiliser la balise <style>. On l'a vu en cours mais on arrêtera après.
