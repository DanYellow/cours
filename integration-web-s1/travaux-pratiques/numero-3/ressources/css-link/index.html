CSS link - Écrivons notre fichier .css — Consignes TP

Consignes de la pratique (cliquer pour afficher / cacher)

  1. Créez un fichier .css, appelé style.css (ou un autre nom), ainsi qu'un fichier .html
  2. Créez des classes CSS et appliquez-les sur des balises HTML

    Vous pouvez aussi utiliser le code que vous avez fait dans l'exercice précédent (css-style) et le transformer avec des classes CSS.

  • Documentation des sélecteurs de classes CSS (ouverture dans un nouvel onglet)
  • Documentation de la balise <link> (ouverture dans un nouvel onglet)
Si vous avez du mal à comprendre les sélecteurs CSS, voici quelques ressources (en anglais) qui peuvent vous aider :

Pour aller plus loin (cliquer pour afficher / cacher)

  1. A partir du site contenu dans le dossier "pour-aller-plus-loin/", externalisez toutes les propriétés CSS contenues dans le fichier HTML dans un fichier CSS

    Vous devez utiliser des classes CSS. N'oubliez pas qu'une classe est réutilisable.
