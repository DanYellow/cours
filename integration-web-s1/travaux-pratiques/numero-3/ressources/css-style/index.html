Consignes de la pratique (cliquer pour afficher / cacher)

  1. Créez un nouveau fichier HTML (ou réutilisez celui de l'exercice "css-balises")
  2. Ajoutez trois balises différentes. Utilisez des balises classiques comme les balises <ul>, <li>, <p> ou encore <a>
    • Liste des balises html (ouverture dans un nouvel onglet)
    Et utilisez-les plusieurs fois, par exemple, 2-3 balises <p> sur la page
  3. A l'aide d'une balise <style>, que vous aurez rajouté dans la balise <head>, appliquez des déclarations CSS sur les balises ajoutées précédemment. Voici un exemple de propriétés CSS que vous pouvez utiliser sur ces balises :
    • Couleur du texte (ouverture dans un nouvel onglet)
    • Bordure (ouverture dans un nouvel onglet)
    • Taille du texte (ouverture dans un nouvel onglet)
    • Décoration du texte (ouverture dans un nouvel onglet)
    • Couleur d'arrière-plan (ouverture dans un nouvel onglet)
    • Documentation de la balise <style>

    Inspirez-vous de la documentation de la balise <style> pour créer vos sélecteurs CSS à mettre dans la balise <style>

  4. Sauvegardez le fichier ( Ctrl/command (⌘) + s ) puis consultez votre page sur votre navigateur

N'oubliez pas qu'on ne peut pas mettre des balises HTML dans une balise <style>.
