Liens TP — Consignes TP

Partie 1 (cliquer pour afficher / cacher)
  1. Créez une nouvelle page html ainsi qu'un fichier css. Liez les deux fichiers
  2. Rajoutez une balise <a> permettant de faire une ancre au sein de cette page. N'oubliez pas l'attribut "href" dont la valeur sera précédée d'un croisillon (#)
    • Voir documentation de la balise <a> (ouverture dans un nouvel onglet)
    • Voir la documentation de la balise <a> d'un lien vers un élément de la même page (ouverture dans un nouvel onglet)
  3. Rajoutez une balise (de votre choix) et mettez-lui l'attribut id dont la valeur sera égale à la valeur de l'attribut "href" de la balise <a> que vous avez précédemment définie

    Note : le défilement ne peut se voir que si votre page est suffisamment haute. Essayez d'appliquer la propriété CSS "height" sur la balise <body> pour afficher une barre de défilement sur la page.

  4. Cliquez sur votre lien. Vous devriez atteindre la cible de votre ancre (la balise que vous venez de définir) et l'url de la navigateur changera

Partie 2 (cliquer pour afficher / cacher)
  1. Ajoutez sur un de vos liens une ou plusieurs pseudo-classes
    • Accéder à la documentation de la pseudo-classe ":visited"
    • Accéder à la documentation de la pseudo-classe ":hover"
    • Accéder à la documentation de la pseudo-classe ":active"
    • Accéder à la documentation de la pseudo-classe ":focus-within"

    Si vous utilisez plusieurs pseudo-classes, attention à leur ordre dans votre CSS pour la balise <a>.

    Afin que la mise en forme s'applique au mieux, la règle :link doit être placée avant les autres : :link — :visited — :hover — :active. La pseudo-classe :focus-within est habituellement placée juste avant ou juste après :hover, en fonction de l'effet désiré.

    — Documentation de la pseudo-classe "link"

  2. Appliquez une pseudo-classe :hover sur un autre type de balise

Partie 3 (cliquer pour afficher / cacher)
  • Partie A
    1. Créez un nouveau fichier .html (avec le squelette d'une page)
    2. Rajoutez un lien (balise <a>) dans ce nouveau fichier html avec comme valeur pour l'attribut "href" le nom de votre premier fichier html
    3. Cliquez sur votre lien. Vous devriez atteindre le fichier html crée au début du TP
  • Partie B
    1. Créez une nouvelle balise <a> (peu importe le fichier html utilisé) au sein d'un paragraphe
    2. Faites pointer cette nouvelle balise vers une url externe de votre choix, une url absolue. Exemple, le site de l'université
    3. Cliquez sur votre lien. Vous devriez atteindre le site que vous avez mis dans l'attribut "href"

N'oubliez pas qu'un lien doit être facilement repérable surtout dans un bloc de texte. Pensez aux propriétés CSS "color", "text-decoration" ou encore "background-color" pour rendre les liens distincts.
