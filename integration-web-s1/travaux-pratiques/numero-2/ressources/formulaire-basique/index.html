Le but de cette pratique est de construire notre premier formulaire en y ajoutant au fur et à mesure des éléments de formulaire tels que la balise <input /> ou encore <select>. Les formulaires sont des éléments clés du web 2.0 et 3.0, grâce à eux, et aux serveurs, l'utilisateur peut communiquer avec le site ou même d'autres internautes. Ainsi, il est promordial d'apporter beaucoup de soin à la construction des formulaires pour éviter de nuire à l'expérience de vos internautes. Sur un bon site, un internaute prend toujours du plaisir à y revenir.

A noter que la gestion des formulaires va de pair avec le cours "développement web" où vous apprendrez à gérer les données en provenance d'un formulaire.

Consignes de la pratique

Partie 1 (cliquer pour afficher / cacher)
  1. Créez une nouvelle page html avec un formulaire (balise <form>)
  2. Ajoutez l'attribut "action" avec la valeur "" pour la balise <form>
  3. Ajoutez du formulaire un nouveau champ <input>
    • Ne pas oublier de rendre le champ accessible. N'oubliez pas :
      • La balise <label>
      • Les attributs associés aux balises et leurs valeurs
  4. Ajoutez dans le même formulaire un autre champ input avec un type "intelligent" (exemple <input type="email">).
    • Ne pas oublier de rendre le champ accessible. N'oubliez pas :
      • La balise <label>
      • Les attributs associés aux balises et leurs valeurs

Le tout doit être dans le même formulaire, pas besoin de créer un formulaire pour chaque champ. Il en est de même pour le reste de la consigne.

  • Voir documentation de la balise <label> (ouverture dans un nouvel onglet)
  • Voir documentation de la balise <input> (ouverture dans un nouvel onglet)
  • Voir documentation de la balise <form> (ouverture dans un nouvel onglet)

Partie 2 (cliquer pour afficher / cacher)
  1. Rajoutez l'attribut "method" avec la valeur "GET" pour le formulaire
  2. Rajoutez un élément de formulaire permettant sa soumission
    • Voir documentation de la balise <button> (ouverture dans un nouvel onglet)
    • Voir documentation de la balise <input> (ouverture dans un nouvel onglet)
  3. Soumettez le formulaire et observez l'url de la page dans votre navigateur

Partie 3 (cliquer pour afficher / cacher)
  1. Rajouter une balise <textarea> avec le label associé et les bons attributs
    • Voir documentation <textarea> (ouverture dans un nouvel onglet)
  2. Rajouter des inputs type radio (min. 2)
    • Voir documentation des boutons radio (ouverture dans un nouvel onglet)
  3. Rajouter des inputs type checkbox
    • Voir documentation des cases à cocher (ouverture dans un nouvel onglet)
  4. Soumettez le formulaire et observez l'url de la page dans votre navigateur
    • Note : Si l'url devient trop longue, n'hésitez pas à la copier et la coller dans VS Code pour l'observer aussi facilement

Partie 4 (cliquer pour afficher / cacher)
  1. Rajouter une liste déroulante simple
    • Voir documentation de la liste déroulante (ouverture dans un nouvel onglet)
    • Outil pour générer une liste rapidement (ouverture dans un nouvel onglet)

Partie 5 (cliquer pour afficher / cacher)
  1. Rajoutez l'attribut "required" sur l'un des champs ajouté précédemment (plusieurs si vous le souhaitez)
  2. Soumettez le formulaire sans remplir les champs que vous avez défini comme requis
