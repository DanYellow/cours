Consignes de la pratique

Partie 1 (cliquer pour afficher / cacher)
  1. Créez une nouvelle page html ou réutilisez votre page de la partie précédente

  2. Créez une balise <img> et lui faire afficher une image locale (mettre l'image au même niveau que votre fichier html)

    • Note : N'oubliez pas de mettre l'extension du fichier. Ceci a une importance capitale
    • Voir documentation de balise <img> (ouverture dans un nouvel onglet)
  3. Sauvegardez le fichier (Ctrl/command (⌘) s) puis consultez votre page sur votre navigateur et pensez bien à actualiser votre page

Partie 2 (cliquer pour afficher / cacher)
  1. Créez un dossier (avec le nom que vous souhaitez) au même niveau que ce fichier index.html
  2. Ajoutez une image dans ce dossier

  3. Créez une balise <img> et lui faire afficher l'image ajoutée précédemment

    • Note : N'oubliez pas de mettre l'extension du fichier. Ceci a une importance capitale
    • Voir documentation de balise <img> (ouverture dans un nouvel onglet)
  4. Sauvegardez le fichier (Ctrl/command (⌘) s) puis consultez votre page sur votre navigateur et pensez bien à actualiser votre page

Partie 3 (cliquer pour afficher / cacher)

  1. Créez une balise <img> et lui faire afficher une image distante (clic droit > copier l'adresse de l'image)

    • Note : Vous pouvez ouvrir le lien dans un autre onglet pour vous assurer que le lien de l'image distante est correct
    • Voir documentation de balise <img> (ouverture dans un nouvel onglet)
  2. Sauvegardez le fichier (Ctrl/command (⌘) s) puis consultez votre page sur votre navigateur et pensez bien à actualiser votre page

Partie 4 (cliquer pour afficher / cacher)
  1. Créez une balise <p> (avec du contenu dedans) puis ajoutez l'attribut "title" avec la valeur textuelle que vous souhaitez
    • Voir documentation l'attribut "title" (ouverture dans un nouvel onglet)
  2. Sauvegardez le fichier (Ctrl/command (⌘) s) puis consultez votre page sur votre navigateur et pensez bien à actualiser votre page (F5 ou la flèche qui tourne vers la droite)
  3. Survolez la balise <p> avec votre souris puis ne déplacez plus la souris, vous devriez voir le texte que vous avez écrit dans l'attribut "title"
