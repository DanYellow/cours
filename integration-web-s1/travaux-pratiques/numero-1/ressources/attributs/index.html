Découvrons les attributs HTML — Consignes TP1

Partie 1 (cliquer pour afficher / cacher)
  1. Créez une nouvelle page HTML ou réutilisez votre page de la pratique précédente
  2. Changez la valeur de l'attribut "lang" de la balise <html> en "fr"
  3. Créez une balise <img> et lui faire afficher une image locale, une image que vous aurez téléchargé. Mettez l'image au même niveau que votre fichier html

    Le HTML peut gérer les formats d'image suivants (liste non exhaustive), chacun de ces formats a des caractérisiques qui lui sont propres. Faites donc attention quand vous les utilisez.

    • JPEG (extension .jp(e)g) : Format le plus populaire. Format statique compressé avec perte. Ne gère pas la transparence
    • PNG (extension .png) : Format statique compressé sans perte. Gère la transparence
    • GIF (extension .gif) : Format animé. Gère la transparence. Limité à 256 couleurs
    • AVIF (extension .avif) : Format animé avec gestion de la transparence. Compressé sans perte et très léger. Géré par les navigateurs modernes

      Le format AVIF n'est pas forcément proposé partout pour la conversion. Le site squoosh.app propose de faire la conversion des images en avif gratuitement.

    • Liste des formats gérés par les navigateurs

    N'oubliez pas de mettre l'extension du fichier. Ceci est indispen Tout comme l'attribut "alt" même s'il doit rester vide.

    • Voir documentation de balise <img> (ouverture dans un nouvel onglet)
  4. Sauvegardez le fichier (Ctrl/command (⌘) + S) puis consultez votre page sur votre navigateur

Pensez bien à actualiser votre page (touche F5 ou l'icone ⟳) après chaque modification de code.

Partie 2 (cliquer pour afficher / cacher)

  1. Créez un dossier (avec le nom que vous souhaitez) au même niveau le fichier html crée dans la partie précédente
  2. Ajoutez une image dans ce dossier
  3. Créez une balise <img> et lui faire afficher l'image ajoutée précédemment

    N'oubliez pas de mettre l'extension du fichier. Ceci est indispensable.

    • Voir documentation de balise <img> (ouverture dans un nouvel onglet)
  4. Sauvegardez le fichier (Ctrl/command (⌘) + S) puis consultez votre page sur votre navigateur

Pensez bien à actualiser votre page (touche F5 ou l'icone ⟳) après chaque modification de code.

Partie 3 (cliquer pour afficher / cacher)
  1. Affichez dans une autre balise <img> une image distante (dans votre navigateur : clic droit > copier l'adresse de l'image)

    Vous ne devez pas télécharger l'image.

    • Voir documentation de balise <img> (ouverture dans un nouvel onglet)
  2. Sauvegardez le fichier (Ctrl/command (⌘) + S) puis consultez votre page sur votre navigateur

Pensez bien à actualiser votre page (touche F5 ou l'icone ⟳) après chaque modification de code.

Partie 4 (cliquer pour afficher / cacher)
  1. Créez une balise <p> (avec du contenu dedans) puis ajoutez l'attribut "title" avec la valeur textuelle que vous souhaitez
    • Voir documentation l'attribut "title" (ouverture dans un nouvel onglet)
  2. Sauvegardez le fichier (Ctrl/command (⌘) + S) puis consultez votre page sur votre navigateur
  3. Survolez la balise <p> avec votre souris puis ne déplacez plus la souris, vous devriez voir le texte que vous avez écrit dans l'attribut "title"
