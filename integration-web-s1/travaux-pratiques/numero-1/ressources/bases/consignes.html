Bases - Notre première page web — Consignes TP1

Partie 1 (cliquer pour afficher / cacher)

  1. Téléchargez et installez le logiciel VS Code (gratuit) - ouverture dans un nouvel onglet

    Si vous êtes sous Windows 7, vous ne pourrez pas installer la dernière version de VS Code, il vous faudra installer la version 1.70. C'est la dernière version à gérer Windows 7.

    • Télécharger la version 1.70 (Cliquez sur "User") - ouverture dans un nouvel onglet

    Si vous utilisez un ordinateur de l'IUT, VS Code est déjà installé.

  2. Créez un dossier de travail sur votre ordinateur (nom au choix)
  3. Glissez-déposez ce dossier dans VS Code
  4. Créez un fichier nommé "index.html" avec VS Code (File > New Text File ou Ctrl / command (⌘) + N)
  5. Écrivez "html:5" puis appuyez sur la touche Entrée. Vous devriez obtenir quelque chose comme ceci : 
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>

</body>
</html>

    Le code ci-dessus représente le squelette de base d'une page web. Sans la base ci-dessus, votre page ne sera pas considérée comme étant valide.

  6. Modifiez le contenu de la balise <title>
  7. Sauvegardez le fichier (Ctrl / command (⌘) + S)
  8. Ouvrez le fichier dans votre navigateur préféré (pas Internet Explorer). Au choix :
    • Glissez-déposez votre fichier index.html avec votre navigateur préféré (pas Internet Explorer)
    • Clic droit sur votre fichier index.html dans l'explorateur Windows / Finder > Ouvrir avec > votre navigateur préféré (pas Internet Explorer)

    Vous ne devez utiliser en aucun cas Internet Explorer. Ce navigateur est déprécié et abandonné par Microsoft. Il n'est plus à jour au niveau des standards du web, ce qui rendra, à coup sûr, un affichage étrange des consignes et autres sites web. Utilisez Chrome, Edge, Opera GX, Firefox ou encore Safari, mais en aucun cas Internet Explorer. Cette note s'applique à l'ensemble des cours que nous aurons tout au long de la formation.

  9. Vous devriez obtenir une page blanche

Quand vous travaillez avec VSCode. Pensez toujours à glisser-déposer votre dossier de travail dans VS Code. Ceci vous permettra d'accéder très facilement à tous les fichiers de votre projet.

Partie 2 (cliquer pour afficher / cacher)

  1. Rajouter une balise <p> avec le texte de votre choix au bon endroit dans le fichier index.html crée précédemment
    • Documentation de la balise <p> - Ouverture dans un nouvel onglet

    Il n'est pas utile d'écrire toute la structure d'une balise. VSCode comporte des raccourcis pour créer des balises, il vous suffit juste d'écrire son nom. Exemple : Ecrire juste "span" puis touche Entrée affichera la structure <span></span>

  2. Sauvegarder le fichier (Ctrl / command (⌘) + S) puis consultez votre dernière mise à jour dans votre navigateur

Pensez bien à actualiser votre page (touche F5 ou l'icone ⟳) après chaque modification de code.

Partie 3 (cliquer pour afficher / cacher)

  1. Rajoutez plusieurs balises <p> avec le texte de votre choix à la suite de balise <p> écrite précédemment

    Sous VS Code, vous pouvez dupliquer la ligne courante vers le haut ou bas avec le raccourci shift + alt / option + ↑ ou ↓

  2. Sauvegardez le fichier (Ctrl / command (⌘) + S) puis consultez votre dernière mise à jour dans votre navigateur

Pensez bien à actualiser votre page (touche F5 ou l'icone ⟳) après chaque modification de code.

  2. Sauvegardez le fichier (Ctrl / command (⌘) + S) puis consultez votre dernière mise à jour dans votre navigateur
