Consignes de la pratique

Partie 1 (cliquer pour afficher / cacher)

  1. Téléchargez et installer le logiciel VS Code (gratuit) (ouverture dans un nouvel onglet).

    Si vous êtes sous Windows 7, vous ne pourrez pas installer la dernière version de VS Code, il vous faudra installer la version 1.70. C'est la dernière version à gérer Windows 7.

    • Télécharger la version 1.70 (Cliquez sur "User") (ouverture dans un nouvel onglet)
  2. Créez un dossier de travail sur votre ordinateur (nom au choix)
  3. Glissez-déposez ce dossier dans vscode
  4. Créez un fichier nommé "index.html" avec VS Code (File > New Text File ou Ctrl/command (⌘) + n)

  5. Ecrivez "html:5" puis appuyez sur la touche "entrée"

    Vous devriez obtenir quelque chose comme ceci :

  6. Sauvegarder le fichier (Ctrl/command (⌘) + s)
  7. Ouvrez le fichier dans votre navigateur préféré (pas Internet Explorer)
    • Glissez-déposez votre fichier index.html avec votre navigateur préféré (pas Internet Explorer)
    • Clic droit sur votre fichier index.html > ouvrir avec > votre navigateur préféré (pas Internet Explorer)
  8. Vous devriez obtenir une page blanche

Quand vous travaillez avec VSCode. Pensez toujours à glisser-déposer votre dossier de travail dans VS Code. Ceci vous permettra d'accéder et voir très facilement tous les fichiers de votre projet.

Partie 2 (cliquer pour afficher / cacher)

  1. Rajouter une balise <p> avec le texte de votre choix au bon endroit dans le fichier index.html crée précédemment

    Il n'est pas utile d'écrire toute la structure d'une balise. VSCode comporte des raccourcis pour créer des balises, il vous suffit juste d'écrire son nom. Exemple : Ecrire juste "span" puis touche entrée affichera la structure <span></span>

  2. Sauvegarder le fichier (Ctrl/command (⌘) s) puis consultez votre dernière mise à jour dans votre navigateur en rafraîchissant la page avec la touche F5

Partie 3 (cliquer pour afficher / cacher)

  1. Rajoutez plusieurs balises <p> avec le texte de votre choix à la suite de balise <p> écrite précédemment. Vous pouvez dupliquer la ligne courante avec le raccourci shift alt / option ↑ ou ↓
  2. Sauvegardez le fichier (Ctrl/command (⌘) s) puis consultez votre dernière mise à jour dans votre navigateur

Partie 4 (cliquer pour afficher / cacher)

  1. Rajoutez plusieurs balises consécutives <span></span> avec le texte de votre choix au bon endroit
  2. Sauvegardez le fichier (Ctrl/command (⌘) s) puis consultez votre dernière mise à jour dans votre navigateur
